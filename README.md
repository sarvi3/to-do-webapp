
This is a minimal, client-side To-Do application that uses localStorage for persistence. It requires no build tools or server — just open `index.html` in your browser.

Features
- Add tasks
- Edit tasks (double-click label or press ✏️)
- Delete tasks
- Mark tasks completed
- Filter by All / Active / Completed
- Clear completed tasks
- Persistence via localStorage

Run / Try it
1. Open `c:\Users\SARVI\Documents\vs\todo-app\index.html` in your browser (double-click or "Open File").
2. Add tasks using the input and press Add or Enter.
